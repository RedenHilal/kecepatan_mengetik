# Typing Game — Java Terminal Application

A fast, responsive typing speed test built in Java using the **Lanterna** terminal UI library.  
Supports multiple game modes, difficulty levels, real-time rendering, and detailed game statistics.

---

## Features

- **Two Game Modes**
  - **Word Mode** — Selesaikan jumlah kata tertentu.
  - **Seconds Mode** — Mengetik selama batas waktu tertentu.

- **Real-time Statistics**
  - WPM (Words Per Minute)
  - CPM (Characters Per Minute)
  - Accuracy %
  - Mistakes count
  - Correct character count

- **Difficulty System**
  - Easy, Medium, Hard, Hell, Code
  - Weighted word generator

- **Terminal UI**
  - Built with **Lanterna**
  - Smooth rendering (no flickering)
  - Custom cursor highlighting

- **Clean OOP Architecture**
  - Inheritance + Interfaces
  - Composition + Aggregation
  - Factory Pattern
  - Modular package structure
